# ✨ Features

## 🌍 City Search

- Search weather by city name
- Supports cities worldwide
- Real-time weather data

## 🌡️ Weather Information

- Current temperature
- Weather condition
- Humidity percentage
- Wind speed
- Feels Like temperature

## 🖼️ Dynamic Weather Icons

- Weather icons loaded dynamically from OpenWeatherMap
- Different icon for different weather conditions

## ⌨️ Enter Key Support

- Press Enter to search weather
- No need to click Search button

## 🔄 Loading State

- Button changes to Loading...
- Prevents multiple API requests

## 📱 Responsive Design

- Mobile-friendly layout
- Responsive weather card
- Optimized UI for different screen sizes

## 🎨 Modern UI

- Glassmorphism design
- Gradient background
- Smooth hover effects
- Clean user experience

# ⚡ JavaScript Functionalities

## API Fetch Request

- Fetches live weather data
- Uses async/await
- Handles API responses dynamically

## Dynamic DOM Updates

Updates UI dynamically:

- Temperature
- City Name
- Weather Condition
- Humidity
- Wind Speed
- Weather Icon

## Error Handling

- Empty input validation
- Invalid city detection
- Network error handling

## Keyboard Events

- Enter key triggers weather search

## Loading State Management

- Disable button while fetching data
- Re-enable after response
